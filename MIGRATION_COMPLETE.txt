================================================================================
HOW TO USE
================================================================================

DEVELOPMENT:
  $ cd backend && ./mvnw spring-boot:run
  $ cd frontend && npm run dev
  
  Access: http://localhost:5173
  All API calls routed via Vite proxy to localhost:8080

PRODUCTION (Co-located):
  $ cd frontend && npm run build
  $ cp dist/* ../backend/src/main/resources/static/
  $ cd ../backend && ./mvnw clean package
  $ java -jar target/edubas-backend-0.0.1-SNAPSHOT.jar
  
  Access: http://yourdomain.com or http://localhost:8080

PRODUCTION (Separate):
  $ export VITE_API_BASE_URL=https://api.yourdomain.com
  $ cd frontend && npm run build
  (Deploy dist/ to CDN)
  
  $ export FRONTEND_URL=https://yourdomain.com
  $ export GEMINI_API_URL=https://generativelanguage.googleapis.com/v1beta
  $ cd backend && ./mvnw clean package
  $ java -jar target/edubas-backend-0.0.1-SNAPSHOT.jar
  
  (Deploy jar to API server)
